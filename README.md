# Trip Cost Calculator

A **Python program** to calculate the total cost of a trip, including hotel stay, flight charges, car rental, and additional spending money. This program is modular and uses separate functions for each cost component.

---

## Features

- Calculates hotel cost based on number of nights.
- Returns round-trip flight cost based on the city.
- Computes car rental cost with applicable discounts.
- Adds optional spending money for an enhanced trip cost.
- Modular and easy-to-understand Python functions.

---
